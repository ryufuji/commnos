# 認証ページデザインガイド

## 概要
Commonsプラットフォームのログイン・登録ページにVivoo風デザインを適用しました。

## 適用ページ

### 1. ログインページ
- **URL**: `/login?subdomain={subdomain}`
- **本番環境**: https://commons-webapp.pages.dev/login?subdomain=test
- **機能**: テナント会員のログイン

### 2. 登録ページ（テナント会員）
- **URL**: `/register?subdomain={subdomain}`
- **本番環境**: https://commons-webapp.pages.dev/register?subdomain=test
- **機能**: テナント会員の新規登録（承認制）

## デザイン仕様

### カラーパレット
- **プライマリカラー**: `#00BCD4` (Cyan)
- **アクセントカラー**: `#FDB714` (Yellow)
- **背景グラデーション**: `#f5f7fa` → `#c3cfe2`
- **装飾シェイプ**:
  - Purple: `#6B4C9A`
  - Terracotta: `#C97865`
  - Cyan: `#00D4E0`
  - Lime: `#7ED957`

### レイアウト構造

#### 背景
- フルスクリーン（100vh）
- グラデーション背景
- 4つの浮遊する有機的シェイプ（opacity: 0.15）
- 20秒サイクルのフローティングアニメーション

#### カード
- 最大幅: 480px（ログイン）/ 520px（登録）
- 背景: 白
- 角丸: 24px
- シャドウ: `0 20px 60px rgba(0, 0, 0, 0.15)`
- パディング: 48px
- フェードインアップアニメーション（0.6s）

### タイポグラフィ
- **タイトル**: 32px, 700 weight
- **サブタイトル**: 16px, 400 weight
- **フォームラベル**: 14px, 600 weight
- **入力フィールド**: 16px

### フォームコンポーネント

#### 入力フィールド
- パディング: 14px 16px
- ボーダー: 2px solid (グレー)
- 角丸: 12px
- フォーカス時:
  - ボーダー: プライマリカラー
  - シャドウ: `0 0 0 4px rgba(0, 188, 212, 0.1)`

#### プライマリボタン
- 幅: 100%
- パディング: 16px
- 背景: プライマリカラー
- テキスト: 白
- 角丸: 12px
- シャドウ: `0 4px 12px rgba(0, 188, 212, 0.3)`
- ホバー時:
  - 背景: `#00a5bb`
  - 上方向移動: -2px
  - シャドウ強化

### レスポンシブ対応

#### デスクトップ (1024px+)
- カード最大幅表示
- 全背景シェイプ表示
- フルサイズアニメーション

#### タブレット (768px-1023px)
- カードパディング調整
- 背景シェイプ維持

#### モバイル (<768px)
- カードパディング: 32px 24px
- タイトル: 28px
- 背景シェイプ非表示（パフォーマンス最適化）

## 使用技術

### CSS
- CSS Variables（commons-theme.css）
- カスタムアニメーション
- Flexbox レイアウト
- メディアクエリ

### フォント・アイコン
- FontAwesome 6.4.0
- システムフォント

### 外部ライブラリ
- Tailwind CSS（CDN）
- Axios（APIリクエスト）

## 実装ファイル

### スタイル
- `/static/commons-theme.css` - グローバルテーマ変数
- `/static/commons-components.css` - 共通コンポーネント
- インラインスタイル - ページ固有のスタイル

### ルート
- `src/routes/tenant-public.ts`
  - `tenantPublic.get('/login', ...)`
  - `tenantPublic.get('/register', ...)`

## 機能

### ログインページ
- メールアドレス・パスワード入力
- ログイン状態保持オプション
- パスワードリセットリンク
- 新規登録リンク
- ホームに戻るリンク
- リアルタイムバリデーション

### 登録ページ

- 自己紹介（任意）

- ログインリンク
- ホームに戻るリンク
- バリデーション
- トースト通知

## パフォーマンス

### 最適化ポイント
- CSS変数によるテーマ管理
- モバイルでの背景シェイプ非表示
- アニメーション最適化（transform使用）
- フォント・アイコンのCDN利用

### ロード時間
- 初回ロード: <1秒
- CSS/JS合計サイズ: 約15KB（gzip前）
- アニメーション: 60fps
